jQuery Hint Plugin Demo

Originally written by Remy Sharp - http://remysharp.com/2007/01/25/jquery-tutorial-text-box-hints/

I improved on it a little bit by adding optional settings and resetting the styles of the input box to its original style when user starts to enter text.

Also now lives in Github so people can fork :)

Demo 1

Watch it disappear:

$("div#demo1 input[title!='']").hint();

Demo 2

With options:

$("input#demo2").hint({color:'green'}); 
    function showNotice() {
      $.n("This is a notice message");
    }
    function showError() {
      $.n.error("This is an error message. It won't go away you have to click the x button.");
    }
    function showWarning() {
      $.n.warning("This is a warning message");
    }    
    function showWithOptions() {
      $.n("This message has options.", {    
        type: "notice",
  			timeout: 5000,
  			stick: true,
  			fadeSpeed : 1500,
  			close : "close me",
  			effect : "slide"})
    }
